三个智能体
灵珑同时接入三个本地 AI，各有特色：
Claude Anthropic 出品
能理解整个项目结构，自主读写文件、执行命令、搜索代码、管理 Git。会话在进程间持续，上下文不丢失。
Hermes Nous Research 出品
自进化的通用智能体，支持技能插件、MCP 协议、定时任务、Slack/WhatsApp 集成。可对接任意 OpenAI 兼容接口。
OpenClaw 开源多通道网关
一个 Agent 对接所有聊天平台：Telegram、Discord、飞书、微信等 20+ 渠道。本地运行，数据不出本机。
点击聊天窗口顶部的标签页切换智能体，各自独立维护对话历史。

开启后不弹气泡通知，适合开会、录屏时使用
🎭
四种工作模式
聊天窗口顶部的模式栏，切换后 AI 的回答风格会变化：
◐ 默认
不注入额外指令，AI 自由回答，适合日常对话
⌨ 编码
代码优先、少废话，直接给可运行的命令和文件路径
🔎 调研
信息整理、多角度对比、列出取舍，帮用户建立认知
⚙ 运维
关注稳定性与风险，操作命令优先给 dry-run 安全形式
切换模式会开启新分支，之前的对话保留在历史中。
🌿
对话分叉与文件引用
对话分叉
点击消息旁的「分叉」按钮，从该点创建新分支。就像 Git 的 branch，你可以从任意对话节点开始新的探索，旧分支随时可以切回。
文件拖入
把文件直接拖进聊天窗口，自动插入 @文件路径 引用，AI 就能读取并处理该文件。
命令面板
在输入框输入 / 触发命令面板，可用 /new 新建对话、/clear 清空历史等。
